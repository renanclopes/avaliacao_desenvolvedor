# Sales Data Importer #
---

## Tecnologias utilizadas
---
* ASP.NET MVC
* Entity Framework 6
* SQL SERVER


## Aplicação
---
Sua utilização tem como objetivo a importação de arquivos de vendas no formato .txt com layout pré-denifido. Os dados importados são gravados no banco e exibidos na página inicial.
